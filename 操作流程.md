# 線上補簽同意書操作流程

本文件說明 Signature-onweb 第二版的實際操作方式。第二版不再依賴 Google Drive 或後端暫存，老師分享的是排版資訊，家長自行載入同意書並完成簽名。

## 一、老師端操作

### 1. 建立排版

1. 老師開啟首頁。
2. 上傳圖片或 PDF 同意書。
3. 檔案上限 5MB。
4. 若是 PDF，目前使用第 1 頁當作簽名底圖。

### 2. 設定簽名欄

1. 點選「新增簽名欄」。
2. 拖曳簽名欄到正確位置。
3. 拖曳右下角把手調整大小。
4. 需要時可下載空白簽名欄預覽圖。

### 3. 產生家長連結

1. 點選「產生家長連結」。
2. 複製系統產生的網址。
3. 將以下兩樣一起傳給家長：
   - 排版好的網址
   - 同意書原檔

## 二、家長端操作

### 1. 開啟排版連結

1. 家長點開老師提供的網址。
2. 系統先顯示簽名欄位置。
3. 系統會提示家長先貼上或上傳同意書。

### 2. 載入同意書

家長可使用以下方式二選一：

1. 直接上傳老師傳來的圖片或 PDF。
2. 若同意書是圖片，也可點選貼上區後直接貼上。

注意：

1. 家長使用的文件最好就是老師傳來的那一份原檔。
2. 若家長使用不同截圖、裁切圖或不同頁面，簽名欄可能偏移。

### 3. 手寫簽名

1. 點選要簽的欄位。
2. 在手寫板簽名。
3. 點選「套用簽名」。
4. 若不滿意可清除重簽。

### 4. 下載完成圖

1. 點選「下載完成圖片」。
2. 系統會輸出 PNG。
3. 若手機未自動下載，可長按預覽圖保存。

## 三、第二版特性

1. 不需要設定 Google Drive。
2. 不需要 OAuth。
3. 不需要後端暫存同意書。
4. 不會自動保存家長文件或簽名。
5. 只要網址可公開開啟，異地家長就能使用。

## 四、目前限制

1. PDF 仍只使用第 1 頁。
2. 完成檔只輸出 PNG。
3. 家長端若使用版面比例不同的文件，簽名位置可能偏移。
4. 第二版尚未支援多位家長的完整流程管理。

## 五、之後可延伸方向

1. 多位家長或多個簽名欄流程。
2. 老師端匯出更完整的分享包。
3. 更穩定的手機貼上與圖片校正。
4. 匯出 PDF 或多格式完成檔。
