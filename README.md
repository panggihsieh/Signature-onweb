# sign

第二版線上補簽同意書工具，分成老師端、家長端與簽名狀態監控頁。

## 頁面

- `index.html`：老師端。上傳、拖曳或貼上同意書，調整簽名欄位位置與大小，產生家長端分享連結與 QR Code。
- `parent.html`：家長端。開啟老師分享的連結後，上傳或貼上同意書，完成簽名並下載或分享到通訊軟體。
- `monitor.html`：老師監控頁。即時查看家長是否已開啟連結、已簽名、已下載或分享完成。

## 啟動

```bash
npm start
```

開啟：

```text
http://localhost:3000/
```

## 檢查

```bash
npm run check
```

## 第二版流程

1. 老師在老師端建立同意書。
2. 老師直接在預覽上拖曳、縮放簽名欄位。
3. 老師按下「複製家長端分享連結」，系統產生家長連結與 QR Code，並開啟監控頁。
4. 家長開啟連結後，上傳或貼上同意書並簽名。
5. 家長下載或分享已簽名 PNG。
6. 老師監控頁顯示家長端狀態。

## 資料保存

第二版不使用 Google 服務，伺服器不保存同意書原始檔。

目前伺服器只暫存簽名狀態監控 session，預設保留 4 小時，可用 `.env` 調整：

```text
TEMP_CASE_TTL_HOURS=4
MAX_UPLOAD_SIZE_MB=5
```
