# 線上補簽同意書 Web App 規劃

本專案目標是建立一個讓老師快速發送同意書、家長線上手寫補簽並下載完成圖片的 web app。

## 專案目標

老師可上傳或貼上同意書，調整簽名欄位後產生家長簽名連結。家長開啟連結後，不需要登入即可在指定簽名欄手寫簽名，最後下載包含「同意書 + 簽名」的完成圖片。

## 角色

### 老師

- 建立同意書。
- 貼上文字內容或上傳同意書檔案。
- 可將同意書暫存到系統設計者提供的 Google Drive 空間。
- 調整簽名欄位位置與大小。
- 產生家長可開啟的簽名連結。

### 家長

- 取得老師提供的連結。
- 不需登入即可開啟同意書。
- 在簽名欄手寫簽名。
- 下載完成圖片，圖片內容需包含同意書與簽名。

### 系統設計者

- 提供一個可寫入的 Google Drive ID 作為暫時儲存空間。
- 設定 Google Drive API 或服務帳號權限。
- 設定暫存檔案自動移除規則。

## 核心需求

1. 不需要登入。
2. 每次上傳同意書大小上限為 5MB。
3. 老師可使用系統設計者提供的 Google Drive 暫存空間。
4. 不建立長期後端資料庫。
5. 整個簽名流程完成後，系統自動移除暫存資料。
6. 家長下載的圖片必須是完整文件圖片，包含同意書與家長簽名。
7. 老師與家長會使用不同裝置完成流程，尤其需支援不同手機開啟同一流程，家長連結必須能跨裝置讀取案件資料。

## 跨裝置使用原則

老師端與家長端不能依賴同一台電腦或同一支手機的瀏覽器本機資料。老師在自己的裝置建立同意書後，家長可能在另一支手機、平板或電腦開啟連結。

因此第一版設計需符合：

- 家長連結只包含案件 ID 或一次性 token。
- 同意書檔案、簽名欄位座標與案件設定必須暫存在後端可讀取的位置。
- Google Drive 暫存資料夾保存原始同意書檔案。
- 案件設定 JSON 需可由家長端透過案件 ID 讀取。
- 不可只把完整同意書資料塞在網址 hash 或 localStorage。
- 完成或過期後，案件 ID/token 失效，暫存資料自動移除。

## 手機使用原則

家長端第一版必須優先支援手機操作：

- 家長可在不同手機瀏覽器開啟簽名連結。
- 簽名板需支援觸控手寫。
- 同意書預覽需能在手機螢幕縮放或適應寬度。
- 簽名欄位在手機上仍需對齊老師設定的位置。
- 下載完成圖片需可在手機瀏覽器觸發。
- 若手機瀏覽器限制自動下載，需至少顯示可長按保存的完成圖片。

## Google Drive 暫存設計

本案需要接 Google Drive，但不是接老師個人的 Google Drive。系統設計者會提供一個 Google Drive folder ID，讓系統暫時寫入同意書檔案。

建議使用方式：

- 後端使用 Google Service Account 寫入指定 Drive folder。
- 老師上傳同意書後，後端將檔案存入暫存資料夾。
- 後端產生一組簽名案件 ID 與家長連結。
- 家長完成簽名並下載圖片後，後端標記案件完成。
- 系統自動刪除該案件相關暫存檔案。
- 若家長未完成，系統依過期時間自動清除，例如 24 小時或 7 天。

## 暫存資料

第一版可暫存以下資料：

| 資料 | 用途 | 清除時機 |
| --- | --- | --- |
| 原始同意書檔案 | 家長端顯示與合成圖片 | 完成簽名後或過期後 |
| 簽名欄位座標 | 家長端知道簽名位置 | 完成簽名後或過期後 |
| 案件設定 JSON | 還原同意書標題、欄位、檔案 ID | 完成簽名後或過期後 |
| 案件 ID/token | 老師與家長跨裝置讀取同一案件 | 完成簽名後或過期後 |

第一版不長期保存：

- 家長簽名。
- 完成圖片。
- 家長個資。
- 老師帳號資料。

## 建議技術架構

- 前端：React、Next.js 或一般 HTML/CSS/JavaScript。
- 後端：Node.js API。
- 暫存：Google Drive 指定資料夾。
- 圖片合成：前端 Canvas 合成後下載，或後端合成後回傳。
- 清除機制：完成後立即刪除，並搭配排程清除過期案件。

## 環境變數

建議使用以下設定：

```text
GOOGLE_SERVICE_ACCOUNT_EMAIL=
GOOGLE_PRIVATE_KEY=
GOOGLE_DRIVE_TEMP_FOLDER_ID=
TEMP_CASE_TTL_HOURS=24
MAX_UPLOAD_SIZE_MB=5
```

## 驗收標準

1. 老師可以上傳 5MB 以內的同意書。
2. 超過 5MB 的檔案會被拒絕並顯示提示。
3. 同意書可寫入系統設計者提供的 Google Drive 暫存資料夾。
4. 老師可在畫面調整簽名欄位。
5. 系統可產生家長簽名連結。
6. 家長可在不同裝置與不同手機不需登入即可開啟連結。
7. 家長可在簽名欄手寫簽名。
8. 家長可下載包含同意書與簽名的完成圖片。
9. 完成流程後，暫存檔案會自動移除。
10. 未完成案件會在期限後自動移除。
11. 家長端在手機瀏覽器可手寫簽名並取得完成圖片。

## 尚待確認

1. 暫存檔案保留時間：建議 24 小時或 7 天。
2. 第一版同意書格式：建議先支援圖片與 PDF，文字貼上可作為第二優先。
3. 完成圖片格式：建議第一版使用 PNG。
4. 是否需要老師端下載空白簽名欄預覽圖。
5. 是否需要同一份同意書給多位家長簽名，或第一版先做一份連結一位家長。
